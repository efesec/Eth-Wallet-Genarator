# 🔵 Ethereum Wallet Generator

Generate Ethereum Mainnet and ERC-20 compatible wallets with ease.

## 🚀 How to Run
1. **Setup:** Ensure Python 3.x is installed.
2. **Install:** `pip install -r requirements.txt`
3. **Run:** `python eth.py`

## 🛠 Technical Specifications
| Feature | Value |
| :--- | :--- |
| **Derivation Path** | `m/44'/60'/0'/0/0` |
| **Standard** | ERC-20 / EVM Compatible |
| **Output File** | `ethwallet.txt` |
